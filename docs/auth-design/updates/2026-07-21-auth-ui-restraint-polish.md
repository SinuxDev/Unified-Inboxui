# Auth polish — less AI, more product UI

**Date:** 2026-07-21

## What changed

- Replaced gradient CTA + shimmer with a solid primary button (`h-11`, flat hover).
- Removed glowing orbs, mesh washes, blue glow on the logo, and uppercase tracking eyebrow.
- Trust panel uses a muted surface + hairline feature list (editorial, not “glow cards”).
- Headline is one color — no blue-highlighted phrase.
- Motion kept purposeful: fade/slide-in on load, feature stagger, feature-row hover, form field stagger. No looping decorative animations.

## Why

The previous treatment read as AI-default (gradients, glow, accent overuse). A UI/UX pass favors hierarchy, restraint, and interaction feedback over ornament.
